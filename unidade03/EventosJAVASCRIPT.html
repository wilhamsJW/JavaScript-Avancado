Tipos de eventos JavaScript

Eventos de navegador:

Eventos Comuns

  • load
    Quando todos os recurso terminam de ser carregados
  • Error
    quando ocorre erro no carragamento
  • Resize
    Qaundo o viweport é rediensionado
  • Scroll
    quando a barra de rolagem é movida

Eventos Dom

  • Focus
    Quando um elçemento recebe foco
  • Blur
    Quando um elemento perde o foco
  • Submit
    qaundo o formulário é submetido
  • Mouse events
    Clique, mouseover, drag e drop
  • onclick
    O usuário clica em um elemento HTML
  • onmouseover
    O usuário move o mouse sobre um elemento HTML
  • onmouseout
    O usuário afasta o mouse de um elemento HTML
  • onkeydown
    O usuário pressiona uma tecla do teclado
  • carregando
    O navegador terminou de carregar a página
